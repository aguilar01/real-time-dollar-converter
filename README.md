💵 Real Time Dollar Converter (Python)
Aplicação desktop desenvolvida em Python que consome uma API de cotação em tempo real para converter Dólar (USD) em Real (BRL).

🚀 Tecnologias utilizadas

Python

Tkinter

CustomTkinter

Requests

API AwesomeAPI

📌 Funcionalidades

✔ Buscar cotação atual do dólar
✔ Converter dólar para real
✔ Calcular valor por item
✔ Tratamento de erros
✔ Interface gráfica moderna

🔗 API utilizada
https://economia.awesomeapi.com.br/

Clone o repositório:

git clone https://github.com/seuusuario/real-time-dollar-converter.git

Instale as dependências:

pip install -r requirements.txt

Execute:

python app.py
